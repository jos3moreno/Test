01 · About

Defense in depth, learned hands-on

I'm a cybersecurity student at Cal State San Marcos working as a Junior Cybersecurity Analyst, focused on the blue-team side of the house: SOC operations, detection engineering, and governance under the NIST Risk Management Framework.

Rather than collecting tutorials, I run corp.lab — a segmented Active Directory environment behind a real OPNsense gateway — where I execute full security cycles: credentialed vulnerability scanning, POA&M-tracked remediation, STIG hardening, and attack simulations detected end-to-end in Wazuh.

RMF Vulnerability Management Cycle

Validated

A complete assess → remediate → validate loop run the way a DoD program would.

  • Credentialed Nessus scans via a dedicated svc-nessus service account
  • Two High findings tracked in a POA&M — CVE-2013-3900 (WinVerifyTrust) and an SQLite disclosure
  • Remediated via GPO "CM-06 WinVerifyTrust Hardening" pushing EnableCertPaddingCheck to both registry paths
  • Closure proven with rescans, documented in corp.lab_POAM.xlsx
