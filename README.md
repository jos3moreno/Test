# jos3moreno.github.io — Security Portfolio

Single-page portfolio (SOC Console theme). No build step — GitHub Pages serves it directly.

## Structure
```
index.html        # markup only
css/styles.css    # all styling (theme variables at the top in :root)
js/main.js        # clock, terminal typing effect, scroll-spy nav
assets/           # resume PDF lives here
```

## Before going live
- [ ] Replace `your-email@example.com` in index.html (2 places — search "EDIT:")
- [ ] Drop resume at `assets/Jose-Moreno-Reyes-Resume.pdf`
- [ ] Point writeup card links at published posts, switch pills from "Draft"
- [ ] Add earned certs (commented template in the Certifications section)

## Deploy
Push to a repo named `jos3moreno.github.io` (root URL) → Settings → Pages → deploy from `main`.
